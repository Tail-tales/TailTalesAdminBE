## TailTalesAdminBE 🐸

**TailTalesAdminBE**는 TailTales 커뮤니티 서비스를 위한 **관리자 전용 백엔드 서버**입니다.  
TailTales는 특수동물을 키우는 사람들을 위한 소통, 정보 공유, 거래 등을 지원하는 커뮤니티입니다.

---

### 🧩 기술 스택

- **Java 17**
- **Spring Boot 3.4.4**
- **Gradle**
- **MariaDB**
- **Lombok**
- **Spring DevTools**
- **JPA/Hibernate**

---

### 🛠 개발 예정 기능
- **관리자 로그인 및 인증 (JWT 기반)**
- **회원/게시글/댓글 관리**
- **커뮤니티 게시판 관리 기능**
- **케어시트**
- **시설 및 리뷰 관리**
